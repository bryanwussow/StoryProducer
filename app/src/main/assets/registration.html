Registration

General:

  1. Registration is encouraged when a local team first begins to use the Story Producer (SP) app, or during the first part of a training workshop.

  2. Filling all the fields and submitting the Registration is very helpful so that SIL Global can support you and continue SP development. (If you are in a sensitive location, enter a pseudonym.)

  3. Tap each information “drawer” to expand it and type in the information. Scroll down to access each field. A few fields are drop down menus which you need to tap on and choose an item. Tap the “drawer” again to close it.

  4. Submit the Registration info via email to SPapp_info@sil.org. When submitting via email, all other emails listed in the Registration fields will automatically be inserted into the email addressee section.

    • If your phone does not have internet connectivity, the Registration info (a small text doc) can be submitted via bluetooth to the Trainer’s phone. Then when the Trainer gets online, they can resubmit or forward the Registration info on to SPapp_info@sil.org.

  1. The Registration information may be edited and (re)submitted when updated information is available.

Language Information:

  • The Ethnologue Code is a 3-letter code which has been assigned to each language. Tap on the WHAT IS THIS? link to access the online Ethnologue database to search for a specific code. If you are working in a language or dialect that has not yet been assigned a code, please type the 3-letter code of the most closely related language followed by a dash (no spaces) followed immediately by 3 primary letters of the dialect name. For example, PEX-MAJ. This code, if inserted and submitted, will appear in the partially automated title of all videos produced so that it is clear with which language the video is produced.

  • Orthography Status has to do with the newness and stability of the language writing system or alphabet. Tap on the drop down menu and then tap on the most appropriate option.

Translator Information:

  • If you wish to enter information for more than one translator, simply put a comma or a slash and list the multiple people with their corresponding information on the same line.

Additional Contacts:

  • This is where other email addresses can be typed if there is an organization or interested party who wants to receive the Registration information.

  • If an organization or interested party would like to receive produced videos, a whatsapp number or an email address could be entered here to provide a way for the local SP team leader to share videos with them.
